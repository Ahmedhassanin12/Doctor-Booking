# Doctor Booking UI Module

## Setup
1. Clone this repository
2. Run `npm install`
3. Run `npm run deev`

## Features
- Doctor directory with filtering
- Booking modal with time slot selection
- Appointments summary view
- Fully responsive design
- Accessibility compliant

## AI Usage
